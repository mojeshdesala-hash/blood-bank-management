# 🩸 Blood Bank Management System
Blood Bank Management System is a Python-based application designed to efficiently manage blood donors and blood stock in a blood bank. The system helps in maintaining accurate records, checking availability, and issuing blood units when required. The application is built using Python with SQLite database, ensuring simple and effective data storage
## 📌 Description
This is a Python-based Blood Bank Management System designed to manage donor details and blood stock efficiently.

## 🚀 Features
- Add Donor Details
- View Donor List
- Manage Blood Stock
- Search Blood Availability
- Issue Blood Units

## 🛠️ Technologies Used
- Python
- SQLite
